# Markdown ✍️
A simple browser based markdown editor, based on React and Redux.

### Features
- Real time preview ✅
- Resizable editor ✅
- Side-by-side editing ✅
- Github-like styling ✅

### Upcoming features
- File saving (WIP)
- Editor preferences
- Toolbar
- Tabs
- Desktop app
